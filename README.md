# Hello1 Repository

Welcome to the Hello1 repository. This repository contains two projects:

## Project 1: Hello World
This project serves as a basic introduction to programming in Python. It prints 'Hello, World!' to the console and showcases fundamental syntax and structures.

## Project 2: Advanced Greeting
This project enhances the basic greeting by allowing users to input their names. It personalizes the greeting, making it interactive and user-friendly.

Feel free to explore the code and contribute!
